# 🇲🇲 Myanmar Civil War - Sitio Informativo

Este proyecto es una página web informativa dedicada a explicar el contexto, evolución y situación actual del conflicto en Myanmar (:contentReference[oaicite:1]{index=1}).

## 📌 Descripción

El objetivo de este sitio es ofrecer información clara, organizada y actualizada sobre la guerra civil en Myanmar, incluyendo:

- Orígenes del conflicto
- Principales actores involucrados
- Línea temporal de eventos importantes
- Situación humanitaria
- Impacto internacional
- Recursos adicionales y fuentes confiables

## 🌐 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (opcional si aplica)
- Frameworks (si aplica: React, Vue, etc.)
